#  Crypto Price Tracker

A live cryptocurrency price tracker built with Python that fetches real-time data from the CoinGecko API.

##  About The Project
Fetches live crypto prices, shows price changes, market caps, and volume data for top cryptocurrencies. Built by a 6-year crypto trader who understands markets deeply.

##  Features
- Live prices for top 10 cryptocurrencies
- 24h price change percentage
- Market cap and volume data
- Price alerts when crypto hits your target
- Historical price data
- Clean terminal dashboard

##  Built With
- Python 3
- Requests library
- CoinGecko API (Free)
-  Requests library
- CoinGecko Free API (no API key required)

##  Installation
pip install requests
python crypto_price_tracker.py
